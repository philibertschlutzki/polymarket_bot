# Archive

This directory contains legacy scripts and backup files that are no longer part of the active codebase but kept for reference.

## Contents

- **backup_env**: Backup of old environment configuration.
- **migrate_to_postgres.py**: One-time script used to migrate from SQLite to Postgres (deprecated now that SQLite is default).
- **verify_db_imports.py**: Old script to verify DB imports.
- **verify_install.py**: Old installation verification script.
- **setup_logrotate.sh**: Old logrotate setup script (functionality now integrated into `deploy_raspberry_pi.sh`).
